# 🌍 Travel Agency Management System

A web-based **Travel Agency Management System** designed to simplify travel package booking, customer management, and tour administration. The system provides an easy-to-use interface for customers to explore destinations and make bookings, while administrators can efficiently manage travel packages, bookings, and customer information.

---

## 📖 Project Overview

The Travel Agency Management System is developed to automate the traditional travel booking process. It allows users to browse available travel packages, check package details, make reservations, and manage their bookings online. The admin panel enables travel agencies to manage packages, customers, bookings, and payments efficiently.

## 🚀 Installation

### Step 1
Clone the repository

```bash
git clone https://github.com/yourusername/travel-agency-management-system.git
```

### Step 2
Move the project folder into the **htdocs** directory.

```
C:\xampp\htdocs\
```

### Step 3
Start

- Apache
- MySQL

using XAMPP Control Panel.

### Step 4
Import the database.

- Open phpMyAdmin
- Create a new database
- Import the SQL file

### Step 5
Update the database connection.

```
localhost
username: root
password:
database: travel_agency
```

### Step 6

Open

```
http://localhost/travel-agency-management-system
```
